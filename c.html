  • Course Code: 2418101
  • Course Title: Programming with C (CSE, AIML)

This course is designed to develop foundational skills in programming using the C language, enabling students to create algorithms, flowcharts, and code effectively.

  • CO-1: Develop flowchart and algorithm to solve problems logically.
  • CO-2: Write simple ‘C’ programs using arithmetic expressions.
  • CO-3: Develop ‘C’ programs using control structures.
  • CO-4: Develop ‘C’ programs using arrays and structures.
  • CO-5: Develop/Use functions in C programs for modular programming approach.
  • CO-6: Develop ‘C’ programs using pointers.

Classroom Instruction: 3 hours/week

Lab Instruction: 2 hours/week

Total Credits: 6

  • Theory Assessment: 30 marks (Internal), 70 marks (External)
  • Lab Assessment: 20 marks
  • Term Work & Self Learning Assessment: 30 marks

Unit 1: Program Logic Development

  • 1.1 Fundamentals of Algorithms
    • Definition of an algorithm
    • Pseudo-code conventions
  • 1.2 Algorithmic Problems
    • Solving arithmetic expressions
    • Finding the greatest of three numbers
    • Determining if a number is even or odd
    • Checking if a number is prime
  • 1.3 Flowchart
    • Understanding flowchart symbols
    • Guidelines for preparing flowcharts

Unit 2: Basics of C Programming

  • 2.1 Introduction to C
    • History of C
    • General structure of a C program
  • 2.2 Data Concepts
    • Character set, tokens, keywords
    • Identifiers, variables, constants
    • Data types and operators
  • 2.3 Basic Input/Output
    • Using printf() and scanf()
    • Input/output formatting

Unit 3: Decision Making and Branching

  • 3.1 Decision Making
    • Relational and logical operators
    • If statements, nested if-else
    • Switch statements
  • 3.2 Looping
    • While, do-while, and for loops
    • Using break and continue statements

Unit 4: Arrays and Structures

  • 4.1 Characteristics of Arrays
    • One-dimensional and two-dimensional arrays
    • Array declaration and initialization
  • 4.2 Structures
    • Introduction to structures
    • Declaration and initialization of structures

Unit 5: Functions

  • 5.1 Library Functions
    • Math functions, string handling functions
  • 5.2 User Defined Functions
    • Writing user-defined functions
    • Parameter passing: call by value and call by reference
  • 5.3 Recursive Functions
    • Understanding recursion
    • Writing recursive functions

Unit 6: Pointers

  • 6.1 Concepts of Pointers
    • Declaring and initializing pointers
    • Pointer arithmetic
  • 6.2 Handling Arrays and Structures with Pointers
    • Accessing array elements using pointers
    • Using pointers with structures
