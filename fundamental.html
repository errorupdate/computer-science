Course Outcomes (COs)

  1. Appraise computer systems and its applications.
  2. Setup a small computer Network.
  3. Write ‘C’ Program to solve given arithmetic expression.
  4. Develop ‘C’ program using control structure.
  5. Develop ‘C’ programs using arrays.
  6. Create functions in C programs for modular programming approach.

Laboratory Experiments

  1. Install device drivers.
  2. Install given software on your system.
  3. Perform registration process of Digital India platform.
  4. Connect two/three computers to form a network using wire/wireless connectivity.
  5. Write and execute simple ‘C’ programs using control structures.
  6. Develop programs using arrays and pointers.
  7. Create user-defined functions in C.

Important Questions

  1. What are the main components of a computer system?
  2. Explain the different types of networks and their topologies.
  3. How do you write a simple C program to solve an arithmetic expression?
  4. What are control structures in C, and how are they used?
  5. Describe the concept of arrays and pointers in C programming.
  6. What is the significance of functions in C, and how do you create them?
