Course Overview

This course provides a foundational understanding of Information Technology (IT) and programming using the C language. It covers essential concepts in computer systems, networking, and programming skills necessary for students to thrive in various educational, business, and industrial domains.

Information technology encompasses the entire range of information generation, storage, transmission, retrieval, and processing. Most organizations now rely heavily on their information systems (IS) and IT. Therefore, students must possess basic skills to use IT and IS effectively.

The C programming language is widely used for developing both basic and advanced applications, making it essential for students to learn programming logic, algorithm development, and program writing.

Course Outcomes

  • Appraise computer systems and their applications in various domains.
  • Set up a small computer network.
  • Write C programs to solve given arithmetic expressions.
  • Develop C programs using control structures.
  • Work with arrays in C programming.
  • Create functions for a modular programming approach in C.

Course Curriculum

Unit 1: Basics of Computer System

This unit covers the anatomy of computer systems, including:

  • Components of a computer system (CPU, memory, storage).
  • Input and output devices.
  • Types of memory (primary and secondary).
  • Communication processes between components.
  • Functionality of computer systems.
  • Introduction to Internet digital platforms (e.g., BHIM, Digi-Locker).

Unit 2: Basic Network Concepts

This unit introduces networking concepts, including:

  • Various computer network topologies (Bus, Mesh, Star, Ring, Hybrid).
  • Types of networks (LAN, WAN).

  • IP addressing system and URL.
  • Functions of networking devices (Switch, Router, Gateway).

Unit 3: Basics of ‘C’ Programming and Control Structures

This unit covers fundamental programming concepts, including:

  • Understanding algorithms and pseudo-code conventions.
  • Flowcharting techniques and symbols.
  • General structure of a C program.
  • Data types, operators, and expressions in C.
  • Input and output statements using printf() and scanf().
  • Control structures: decision-making (if, switch) and looping (for, while).

Unit 4: Arrays and Pointers

This unit focuses on:

  • Characteristics and operations on one-dimensional and two-dimensional arrays.
  • Pointer concepts: declaration, initialization, and accessing memory locations.
  • Manipulating arrays of characters and strings.

Unit 5: Concept and Need of Functions

This unit discusses:

  • Library functions and their usage.
  • Creating user-defined functions and understanding scope of variables.
  • Parameter passing techniques: call by value and call by reference.
  • Recursive functions and their applications.
