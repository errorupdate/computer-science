Fundamental of Computer Science

Computer

  • The word computer comes from the word "Compute", which means, "to calculate"
  • Computer is an electronic device that can be programmed to perform arithmetic and logical operations
  • A Computer is also called a data processor because it can store, process, and retrieve data

Here are some key components and concepts related to computers:

  1. Hardware

    • Central Processing Unit (CPU)

    • Memory (RAM)

    • Storage

    • Input Devices

    • Output Devices

  2. Software

    • System Software

    • Application Software

  3. Data Processing

  4. Networking

  5. Types of Computers

    • Personal Computers (PCs)

    • Servers

    • Tablets and Smartphones

    • Supercomputers

  6. Applications
