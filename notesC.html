1.1 Fundamentals of Algorithms

1.1.1 Notion of an algorithm

An algorithm is a step-by-step procedure or formula for solving a problem. It is a finite sequence of well-defined instructions to achieve a specific goal or perform a computation. Algorithms are fundamental to computer science and programming, as they provide a clear method for solving problems.

  • Finiteness: An algorithm must always terminate after a finite number of steps. It should not run indefinitely.
  • Well-defined Inputs and Outputs: An algorithm should have clearly defined inputs (data it receives) and outputs (results it produces). The inputs can be zero or more, and the outputs can also be zero or more.
  • Effectiveness: Each step of the algorithm must be basic enough that it can be performed, in principle, by a person using only paper and pencil. This means that the operations must be feasible and not require advanced technology.
  • Generality: An algorithm should be applicable to a broad set of problems, not just a single instance. It should be able to handle different inputs and produce the correct output for each.
  • Unambiguous: Each step of the algorithm must be clear and unambiguous, leaving no room for interpretation.

1.2 Algorithmic Problems

1.2.1 Importance of Algorithm

An algorithm is a step-by-step procedure or formula for solving a problem. It is a finite sequence of well-defined instructions to achieve a specific goal or perform a computation. Algorithms are fundamental to computer science and programming, as they provide a clear method for solving problems.

  • Efficiency: Algorithms help in optimizing the use of resources (time and space). A well-designed algorithm can significantly reduce the time complexity of a problem.
  • Problem Solving: Algorithms provide a systematic approach to solving problems, making it easier to understand and implement solutions.
  • Reusability: Once an algorithm is developed, it can be reused in different programs or applications, saving time and effort.
  • Foundation of Programming: Understanding algorithms is crucial for programming, as they form the basis for writing efficient and effective code.
  • Communication: Pseudo-code and flowcharts allow programmers to communicate their ideas clearly with others, regardless of the programming language they use.
